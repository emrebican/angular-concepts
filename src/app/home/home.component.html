Components

Smart & Dumb Component

In Angular, a "smart" component is a component that is aware of and manages data, whereas a "dumb" component is a presentational component that simply displays data without any awareness of or control over it.

Smart components are typically responsible for managing application state and logic, while dumb components are responsible for rendering the user interface based on the data provided to them by the smart components.
